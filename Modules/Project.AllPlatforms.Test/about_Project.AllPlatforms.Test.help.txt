
TOPIC
	about_Project.AllPlatforms.Test

SHORT DESCRIPTION
	Functions to write powershell unit tests
	Functionality for firewall rule auditing

LONG DESCRIPTION
	These simple function are designed for Windows Firewall Ruleset

EXAMPLES
	To start unit test:
	Start-Test

	Before starting a new test case:
	New-Test

	To end test:
	Exit-Test

	To query weak rules with no LocalUser value
	Test-PrincipalRule
