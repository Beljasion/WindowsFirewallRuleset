SHORT DESCRIPTION
	Test if environment is compatible for Windows Firewall Ruleset project

LONG DESCRIPTION
	Requirements for Windows Firewall Ruleset project are:

	1. Windows 10 Pro/Enterprise
	2. Windows Powershell Desktop 5.1
	3. NET Framework 4.8
	4. Administrator account
	5. TCP IP NetBios helper service is started

VARIABLES
	RepoDir:    Root directory to WindowsFirewallRuleset

EXAMPLES
	To make a test:
	Test-SystemRequirements $true
